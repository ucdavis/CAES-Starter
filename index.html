CA&ES kick-off template

Use this document as a way to quickly start any new project.
All you get is this text and a mostly barebones HTML document.

To-do list

  • Decide which libraries you want or don't want, and comment the unnecessary ones out in _vendors.scss, config.rb, and index.html.
  • Set up Google Analytics and change your account number in the footer of index.html.
  • Set up the typography of the site with a new Typekit kit if needed.
  • Fill out humans.txt with relevant credits.

Handy things

You might want to install some of the following items.

  • Sublime Text 2 — SASS/SCSS syntax highlighting
  • Sublime Text 2 — Ruby/HAML syntax highlighting
